# GifApp

Basic application for searching Gif


Technical Info:
IDE: Xcode ( Version 11.3.1)
Development Language: Swift verion 5.0 
OS : ios 13.0
FrameWork Used : Kingfisher (for gif and image handling)
Architecture : MVVM 
API : Giphy (used for getting GIF)
